# API_SOLICITACOES_NEXUS
A Página de Solicitações do Portal Nexus foi criada para centralizar e padronizar as demandas do time, trazendo mais organização, visibilidade e agilidade no tratamento dos pedidos. Com ela, aplicamos boas práticas de atendimento e acompanhamento, reduzindo retrabalho e melhorando a experiência de quem solicita e de quem atende.


# API Nexus – FastAPI

API desenvolvida para gerenciamento de solicitações, upload de arquivos e integração com o NOC.

## 🚀 Funcionalidades

- Criar solicitações
- Finalizar solicitações
- Buscar solicitações por período
- Listar áreas solicitantes
- Upload de arquivos por solicitação
- Listar arquivos anexados

## 📌 Tecnologias

- Python 3.10
- FastAPI
- SQLAlchemy
- MySQL
- Pydantic v2
- Uvicorn

## ▶️ Rodando localmente

bash
pip install -r requirements.txt
fastapi run main.py
